# Creating a repo
* git init
* git add `file-name`
* git commit -m "comment"
* git pull `repo-name` `branch-name`
* git push `repo-name` `branch-name`
* git remote set-url origin git@github.com:username/repo.git



## Remove a repo
git remote remove `repo-name`
